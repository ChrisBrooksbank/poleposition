# Pole Position

Research and recreation of the classic 1982 Namco Pole Position arcade racing game.

## Status

Early research phase. See [research.md](research.md) for notes on the original game mechanics, rendering techniques, and implementation approach.

## About Pole Position

Pole Position (1982) was a groundbreaking arcade racer featuring a pseudo-3D perspective using sprite scaling, a real Fuji circuit track, and qualifying laps before the race. It was one of the first games to use a realistic racing circuit.
